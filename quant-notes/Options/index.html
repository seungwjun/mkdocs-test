    Options

    Option Greeks

    • Delta ($/$). Can be viewed as the probability the option expires in the money. The number is positive for calls and negative for puts.
    • Gamma (1/$). The derivative of the delta. The number is always positive.
    • Theta ($/day). T is for time. How much does the option lose the value in a day? The number is always negative.
    • Vega ($/%). V is for volatility.
    • Rho. R is for rate. Has to do with interest rate. Usually ignored for short-horizon options.
