    Generator

    A generator function is a function that uses yield. Calling such a function returns a generator object.

    The generator object works like an iterator.

    • You can call next(g). Each call returns an item.
    • When it reaches the end, it throws a StopIteration exception.
    • The generator reaches the end if it (1) reaches the end of the function, (2) returns (if return returns a value, it's an error), or (3) raises a StopIteration exception.

    If you call another generator within a generator, use yield from.
